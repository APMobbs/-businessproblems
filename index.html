    How can we overcome issues people with disabilities may face when using webpages?

    Many technological devices and systems have been desgined to help allow people with disabilities be able to enjoy the wonder of computers and the online world. These systems and technological can be divided into a few types; Assistive Software, Keyboard Input Modifications and Assistive Devices, here are a few examlpes of each type:

    Assistive Software

  • Optical character recognition (OCR) - This software is desgined to scan printed materials directly into the PC, this accommodates many types of disabilities.
  • Screen readers - Screen readers are desgined to help the people who are blind or visually impaired using braille or by speaking the words on screen.
  • Magnification software - This software helps by enlarging text and graphics displayed on a computer monitor this is widely used by persons with poor vision or who have difficulty reading.

    • Keyboard Input Modifications

  • Adapted keyboards - there are many kinds of adapt keyboards, some with bigger keys, some with simpler layouts and others that are a different shape, but all help in their own way
  • Voice recognition - this device is less of a keyboard but functions in place of one by recognising a user's voice and typing it out on screen
  • Alternative communication programs - their is a large range of Alternative communication programs that understand different types of communication outside of verbal or writen, they use gesutures or Sign Language

    • Assistive Devices

  • Sip-and-puff systems - This device can be activated thougth the user inhaling or exhaling
  • Trackballs - A Trackball is a small movable balls on top of a base that can be used to control the cursor on screen
  • Joysticks - Joysticks similar to those found in cars can be designed to work with computers, they can be contolled by hand, feet, chin, etc. and used to move the cursor on screen
